## 部分功能已经开通接口

### 签到

``` python
 # url:https://o.nvidia.fun:12345/sign
 # method:post
 # parmas:
{
    "phonenums": "手机号",
    "key":"密码"
}

 # response
{
    "msg": "不在签到时间内无法签到",
    "value": "True"
}
```

### 离开、退座和取消

* 使用方法与签到一致，离开、退座和取消分别把url上的"sign"替换成"leave"、"signback"和"cancel"即可~
